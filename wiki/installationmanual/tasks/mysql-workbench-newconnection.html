Creating a New Connection

This section describes the process for creating a new connection in the MySQL Workbench.

  1. In the home screen of the workbench, please click on the New Connection button as shown below.

  2. In the window that opens up, please specify the connection name as 'Snofyre' and click Test connection button as shown below.

  3. In the dialog that opens up, please enter your MySQL 'root' password. You may optionally chose to store your root password by checking the 'Save password in vault' option as shown below.

    The workbench now stores the connection for future use as shown below.
