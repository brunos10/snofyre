System Requirements

This section describes the system requirements required to run Snofyre on a client machine.

Hardware Requirements

The following is a list of minimum hardware requirements of the client machine.

  • Intel PC (x86 architecture)
  • Dual Core processor 2.0 GHz or higher (recommended)
  • 512 MB RAM (minimum), 1 GB recommended
  • 2GB of free hard disk space (minimum)
Software Requirements
  • Operating System : Windows XP SP3, Mac OS X 10.5 or higher
  • Java Run Time Environment (version 1.5 or later)
  • MySQL Community Server version 5.1 or higher.
