Snofyre Installation Manual - Draft

Getting Started

This chapter describes the installation procedure and system requirements for installing and running Snofyre on a client machine. If you require help installing Snofyre, please send a mail to Snofyre Users Group at snofyre-users@googlegroups.com
Tip: You can subscribe to the Snofyre User Group by sending an email to snofyre-users-subscribe@googlegroups.com
Attention: This documentation has not been updated to reflect Snofyre's final name. Some of the illustrations in the document might still display 'Sappheiros'. Sappheiros was Snofyre's in-house name during development.
Note: Please note that Snofyre and its installer requires Java Run Time Environment to be installed on the client machine. The application can not be run using Java Web Start. Please go to http://www.java.com/en/download/manual.jsp to download and install Java on your machine.
Important: Please note that Snofyre requires an instance of MySQL server to store data and execute queries against. This manual describes the steps for installing and configuring MySQL server.
Since installing and configuring MySQL server is critical for Snofyre, to work, a large part of this manual is devoted to this process. The rest of this manual is broken into the following sections:
  • Snofyre Initial Installation Steps
  • Installation and configuration of MySQL Server for Snofyre
  • Snofyre Final Installation Steps
  • System Requirements
  • Reporting Issues
This manual is still being updated. We apologise for illustrations that still display Snofyre's developmental name (Sappheiros).
