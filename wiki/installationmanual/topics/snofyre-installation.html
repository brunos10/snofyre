Snofyre Installation Overview

This chapter provides an oveview of installing Snofyre on a client machine that meets the System Requirements.

Attention: The details of the installation of the Snofyre and its configuration described here apply to version 1.8. It is likely that these details might change in the future.
Attention: This chapter also assumes that the user has already Java Run Time Evironment installed on their machine. For any issues with installing these software, please refer to the platform specific instructions on http://www.java.com.

It is possible to install and run Snofyre in a Client mode or a Server mode.

Client Mode

In this mode only files necessary for running the Snofyre as a client are installed. This mode only requires the Sun Java Run Time to be installed on the user machine. It does not require installation of a MySQL server on the user machine. In this mode, the SNOMED content and test data required for running Snofyre are located on a remote server machine. This data is accessed by Snofyre during initialisation and when queries are executed. This installation mode will not be described in the configuration manual since Snofyre version 1.8 does not support this installation mode.

Server Mode

In this mode all files necessary for generating data and executing queries from Snofyre are installed on the local machine. This mode requires installation of a local MySQL server. The MySQL server will then need to be configured before it can be populated with SNOMED content and data generation. The remaining sections of this manual describe the process for installing and configuring MySQL on the local machine for Snofyre to use.

The following are the steps for obtaining and installing Snofyre on a client machine.
  1. Download a copy of the Snofyre installer from the download location. The download location for version 1.8 is accessible only within the NHS Connection for Health network. The installer is located at the network location S:\SRTHImpl\Snofyre\Installers
  2. The following is the directory structure of the download location. 
     Snofyre
    |__ Snofyre-mysql-configuration.pdf
    |__ ReadMe.txt
    |__ Installers
            |__ Snofyre-Fat-Installer.exe
            |__ Snofyre-Fat-Installer.jar
    In the future, Snofyre might be made available for dowload outside the CFH network.
  3. Download the file SRTHApp.zip from the above location to a location of your choice.
  4. Once the file has been successfully downloaded, please run the installer and follow the steps to install Snofyre to a folder on your local machine. From now on, we will refer to this folder where you have installed Snofyre as the Installation Folder. On Windows, we recommend that the the folder is extracted to C:\Program Files. This will result in a new folder called C:\Program Files\Snofyre being generated after installation.
