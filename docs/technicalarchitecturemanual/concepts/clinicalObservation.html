Clinical Observation Entity

An interface representation of an 'clinical observation' as asserted in the clinical record. It differs from a Clinical Finding entity which is also a 'clinical observatio' in that a Clinical Observation enity refers to the 'clinical observation' described as a SNOMED CT Observable Entity associated with a numerical value.

Hemoglobin Concentration 9 gm/dL is a Clinical Observation Entity because it is the combination of a the SNOMED Observable Entity 'Hemoglobin Concentration' with a numerical value 9 gm/dL.
Note: In practice, it is not always possible to find a SNOMED CT Observable Entity for every representing a Clinical Observation Entity. In such cases, it might be necessary to use a corresponding Clinical Finding (or Procedure) type SNOMED CT concept in association with a numerical value. In order to accomodate the incomplete modelling of Observable Entities in SNOMED CT, Snofyre does not enforce a very tight coupling with a SNOMED CT entity type when creating a 'Clinical Observation Entity'.
Note: A developer using the API to create a Clinical Observation Entity should note that Snofyre API generates a warning when a SNOMED CT concept type other than an Observable Entity is used to instantiate a Clinical Observation Entity object. The developer is expected to handle this warning appropriately.
