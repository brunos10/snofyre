Application Listener

An Application Listener is a wrapper around a Listener in the traditional desktop application programming paradigm.

This allows the application developer to:
  • Propogate events to the Snofyre's modular components in a thread-safe manner.
  • Enable or disable the Action Components or View Component depending on the availability of an underlying Snofyre service.
