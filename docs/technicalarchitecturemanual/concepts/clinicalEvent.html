Clinical Event

An interface that represents an event entry asserted in the clinical record. It is a special kind of Clinical Entry that has a Status associated with it. For this reason, it is guaranteed to have all the attributes of a Clinical Entry, but also have an additional attribute Status. The Status attribute describes if the clinical event has been planned, completed or aborted.

The clinical entities in Snofyre's information model that correspond to a Clinical Entry are interventions, investigation and medications.
