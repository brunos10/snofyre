View Component

A ViewComponent is a semi autonomous visual component of Snofyre that can be considered an application window in its own right. This allows the view component to respond to changes in underlying Snofyre's services without affecting any other visual component in the main application window.

This allows the user to:
  • Change the layout of the application interface to suit their needs.
  • Hide,close or maximise an interface component to use the real estate on the interface more efficiently.
  • Operate the component in a Floating mode which allows the component to be displayed in a window separate from the main application window.

Within Snofyre, view components typically exist as Docked Panels.
