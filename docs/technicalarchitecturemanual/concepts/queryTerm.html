Query Term

A query term is the SNOMED CT encoded 'token' which the user specifies as the criterion for retrieving clinical records/entries that match the 'token'. A Query Term can either be a SNOMED CT concept ID or a SNOMED CT encoded clinical expression.

Note: Typically users associate concept ids and text strings with query terms, but in a post-coordination enabled system,it might not always be possible to associate a single concept id or text string with a Query Term.
The following are all valid examples of a Query Term:
  • 22298006|Myocardial Infarction (disorder)|
  • 222 ...
Subsumption Strategy
The boundaries of a 'search token' can be defined using the Subsumption strategy property which allows the user to specify the depth in the concept hierarchy that needs to be traversed while returning matches.
SELF_ONLY
Specifies that only the 'search token' included in the Query Term is to be used to return matches.
ALL_CHILDREN
Specifies that the 'search token' in the Query Term and all its child concepts/expressions are to be used to return matches.
ALL_DESCENDANTS
Specifies that the 'search token' in the Query Term and all its descendant concepts/expressions are to be used to return matches.
ALL_CHILDREN_EXCEPT_SELF
Specifies that all the children of the 'search token' in the Query Term but not the 'search token' itself are to be used to return matches.
ALL_DESCENDANTS_EXCEPT_SELF
Specifies that all the descendants of the 'search token' in the Query Term but not the 'search token' itself are to be used to return matches.

Query | Get all patients with 'Myocardial Infarction' | typically returns all patients who have a mention of 'Myocardial Infarction' in their record. However, this query typically will not return patients with 'Acute Q Wave Myocardial Infarction' even though "Acute Q Wave Myocardial Infarction' is a type of 'Myocardial Infarction'. In fact, SNOMED CT lists 75 types of 'Myocardial Infarction' and ideally, one would expect the afore mentioned query to return any patient who had any of the 75 types of 'Myocardial Infarction'.
