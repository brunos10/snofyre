SNOMED Reporting Test Harness

The SNOMED Reporting Test Harness is an application framework that allows the authoring of clinical queries described using SNOMED CT concepts, management, transformation (typically to SQL) and the execution of these queries against a data store (typically implemented as a relational database).

The SNOMED Reporting Test Harness has the following components.
  1. SNOMED Reporting Test Harness Application Programming Interface
  2. SNOMED Reporting Test Harness User interface

    The User Interface is the graphical user interface via which the user typically interacts with the SNOMED Reporting Test Harness.

    The Application Programming Interface API is implemented in JAVA and the corresponding documentation will be included as part of the standard distribution. This allows a developer to use the SNOMED Reporting Test Harness within their own application.
