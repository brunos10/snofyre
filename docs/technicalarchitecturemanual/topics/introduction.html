Snofyre Technical Architecture Manual - Draft

Introduction

Welcome to the SNOMED Reporting Test Harness. This manual explains how to use the SNOMED Reporting Test Harness to quicky test cases for clinical reporting and execute them against a test data store.

This manual assumes that the SNOMED Reporting Test Harness user has some familiary with relational database technologies, SQL and SNOMED CT.

What is the SNOMED Reporting Test Harness

The SNOMED Reporting Test Harness is an application framework that allows the creation of clinically queries described using SNOMED CT concepts and the execution of these queries against a data store (typically implemented as a relational database).
