Snofyre Technical Architecture Manual - Draft

How this document is organised

This document helps the reader understand how Snofyre was built and how it works. This document includes the following sections.
Important: This document does not explain how all the details of how Snofyre's underlying services work. These are described in the Snofyre Functional Services Manual. So, if you are interested in reusing the core services of Snofyre, then refer to this document and the Functional Services Manual.


                        Introduction
                        This section provides an overview of what Snofyre is, the motivation behind its creation and information on licensing restrictions.
                


                        Getting Started
                        This section tells you what this document covers and does not cover, who it is aimed at and relevant information about using Snofyre.
                


                        Application Framework Overview
                This section describes how Snofyre is built from ground up and how its various components relate to each other.
                

Snofyre Services Overview
                        This section describes what functional services (reusable components) Snofyre implements and how they related to each other


                        Process Overview
                        This section describes what Snofyre can do and how it does what it does under the hood.
                


                        Design Overview
                        This section describes how Snofyre represents and stores information behind the scenes.
                


                        Appendix
                        An appendix of all relevant documents.
