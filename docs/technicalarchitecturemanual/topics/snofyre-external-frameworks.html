Snofyre External Frameworks

In building Snofyre we used a large number of popular third party frameworks and libraries. The following is a list of important external frameworks that have been used.

External Frameworks

  • OSGi for modularity
  • Spring and Spring DM for dependency injection
  • Hibernate for persistence of data.
    Important: Snofyre does not always use Hibernate for querying. This means that the Query Transformer service generates SQL for a given Snofyre query and not HQL (Hibernate Query Language).
  • Lucene for lexical search of SNOMED CT content
    Important: Snofyre does not use SQL to search SNOMED CT content.
  • JUnit for all unit tests.
    Important: If you extend Snofyre API and want to contribute code, please include test cases!
