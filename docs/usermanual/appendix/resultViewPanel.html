Query Results Panel

This is the part of Snofyre's graphical user interface displays the results of the execution of a query against the test data.

This panel is usually located at the top right hand side of the application.

It contains two sections:
  • Query Statistics
  • Results Table
Query Statistics
This section of the panel displays information about
  • Number of patients satisfying the criteria specified in the query.
  • Query Time : The time needed to execute the query against the test data and display the results in the Results View Panel. Please refer to Query Time for more information.
    Note: The time take to return the results set from the backend might be slightly less than the time needed to return the result set from the database and render in the table in the graphical user interface.
Results Table

This section of the panel displays the results of query execution for visual inspection by the user. The table in general displays the following information.

  • Patient Id : The id of the patient who satisfied all the criteria defined in the query.
  • Free Text Entry : The free text entry corresponding to the necessary criteria in the query.
  • Entry Time : The entry time for every entry corresponding to the necessary criteria in the query.
  • Concept ID : The concept ID or UUID (if present) that corresponds to the necessary criteria that satisfy the query.
Important: Snofyre hides the Entry Time and Concept ID columns in the Results Table by default.
Tip: You can sort the contents of the columns in the Results Table, by clicking on the column name. This is useful when you need an alphanumeric sorting of the free text entries.

Please refer to Interpreting Results for more information on 'necessary criteria' and the the columns included in each result set.
