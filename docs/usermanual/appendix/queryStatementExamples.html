Query Statement Examples

The section provides examples of query statements that can be used in the Snofyre.

Query | All patients with Diabetes | is query that contains a single Query Criterion (with a query term of 'Diabetes')

Query | All patients with Diabetes AND Hypertension | is a query that contains three query criteria;
  1. QueryCriterion (with query term of 'Diabetes')
  2. QueryCriterion (with query term 'Hypertension')
  3. Intersection Expression that contains QueryCriterion objects #1 and #2
Query | All patients with 'Diabetes' AND (Renal Disease OR Heart Disease) | contains 5 reporting expressions.
  1. QueryCriterion with query term 'Diabetes'
  2. QueryCriterion with query term 'Renal Disease'
  3. QueryCriterion with query term 'Heart Disease'
  4. Union Expression containing QueryCriterion objects #2 and # 3
  5. Intersection Expression containing QueryCriterion #1 and Union Expression #4
Query | All patients who have (Radiotherapy to axilla (procedure) and all descendants AND Excision of breast tissue (procedure) and all descendants AND (History of malignant neoplasm of breast (situation) and all descendants OR Malignant tumor of breast (disorder) and all descendants))| contains 6 reporting expressions.
  1. QueryCriterion with query term 'Radiotherapy to axilla (procedure)'
  2. QueryCriterion with query term 'Excision of breast tissue (procedure)'
  3. QueryCriterion with query term 'History of malignant neoplasm of breast (situation)'
  4. QueryCriterion with query term 'Malignant tumor of breast (disorder)'
  5. Union Expression containing QueryCriterion expressions #3 and #4.
  6. Intersection Expression containing
    • QueryCriterion #1
    • QueryCriterion #2
    • Union Expression #5
