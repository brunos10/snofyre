Creating an OR Query

This section describes how to create a new OR query in Snofyre.

  1. A new OR query can be created in any of the following ways:
    • Select File > New Query.
    • Click the New Query button on the toolbar.
    A stub entry for a new query is created in the Query Authoring Panel and this is reflected in the Query Rendering Panel. This will clear any query that has already been loaded in a previous step or an unsaved query.
    CAUTION:
    This action will result in the loss of any modificiations that the user might have made to an existing query, already loaded.
  2. Right click Get patients who have stub in the Query Authoring Panel. The same effect can be achieved by a combination of Crtl button and left click on the Windows platform and Cmd button and left click on the Mac platform. This results in appearance of the query creation pop up menu shown below.

  3. In the query creation pop up menu, move/hover your mouse over Add Logical query as shown below.



    This will expand the menu to show two sub menus as shown below.

  4. In the query creation pop up menu, click on ANY of the following as shown below.



    This updates the Query Authoring Panel and displays the ANY of the following stub as shown below.

  5. Right click the ANY of the following stub and select Add Query Criterion and continue as described in Creating a Simple Query to add a query criterion to this OR query. Snofyre will add the newly created query criterion to this OR query.
  6. Repeat this process as necessary to add more query criteria to the OR query.
