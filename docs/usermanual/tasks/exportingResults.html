Exporting Results

This task describes the steps for export the results returned following the successful execution of a query.

  1. Select Query > Export Query Results. This step opens a window for choosing a location and file name to export the results to.
  2. Specify file name and location The window opened in the previous step allows the name and location of a file where the results are to be exported to be specifed.
    1. Specify the location to save the file to.
    2. Specify the file name to export the results to.
    3. Click Save to export the results to file selected.
    Snofyre now exports the results of the current query to the specified file in a tab delimited format.
