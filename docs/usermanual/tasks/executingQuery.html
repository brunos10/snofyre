Executing a Query

This task explains the process for executing a newly created query (or one that has been loaded from a file). This query is referred to as an Active Query.

An Active Query can be executed by:
  • Select Query > Run Query.
  • Click the Execute Query button on the toolbar.
Note: Once query execution is initiated, it can take a long time for the query execution to complete. Since it is often impossible to predict the query execution time beforehand, Snofyre displays a busy button in the bottom right corner of the application window, while the query is being executed.
This results in the active query being executed against the database. After the query has been executed, the results are displayed in the Results View Panel.
