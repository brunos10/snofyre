Viewing Results

This tasks dessribes the steps to be followed to visually inspect the results returned after executing a query in Snofyre.

  1. Create a new query or load an existing query.
  2. Execute the query. After the successful execution of every query, the ids of all record entries matching the criteria specified by the query are displayed in the Query Results Panel automatically as shown below.



    Note: A summary of the number of records that satisfy the criteria and the time taken to return the results is also displayed by default.
    Tip: You can sort the contents of the columns in the Results Table, by clicking on the column name. This is useful when you need an alphanumeric sorting of the free text entries.
