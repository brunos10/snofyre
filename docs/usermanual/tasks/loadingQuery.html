Loading a Query

This section describes how to load a query that has been saved to a physical location on the client machine.

  1. A query can be loaded in any of the following ways:
    • Select File > Open Query.
    • Click the Load Query button on the toolbar.
    This step opens a window where the user can navigate to the location of the query and select the query file to load.
  2. Navigate to the location of the saved query and choose the query to load, in the window that opens.
    • You can then load the query by double clicking on the file in the window
    • You can load the file by selecting the file and clicking the Open button in the window.
    Snofyre will load the query and display it in theQuery Authoring Panel and the Active Query Panel.
