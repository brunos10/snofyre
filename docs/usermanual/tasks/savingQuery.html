Saving a Query

This section describes how to save a new or modified query to a physical location on the client machine.

  1. A query can be saved in any of the following ways:
    • Select File > Save Query.
    • Click the Save Query to File button on the toolbar.
    This step opens a window where the user can navigate to the location of the query and select the query file to load.
  2. The window opened in the previous step allows the name and location of a file to be specifed.
    1. Specify the location to save the query to.
    2. Specify the file name to store the query as.
    3. Click Save to save the query to file selected.
    Snofyre will save the query to the specified location, as an XML document.
    Tip: We recommend saving queries to a file with a xml extension, so they can be opened easily in any other XML editor.
