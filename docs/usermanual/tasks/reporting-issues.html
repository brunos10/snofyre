Reporting Issues

This section describes the recommended process for reporting bugs or issues encountered when installing or using Snofyre.

  1. When you encounter an unexpected behaviour when installing or using Snofyre, use either of the following mechanisms to open the bug report form.
    • If Sappheirs has crashed, open your browser and navigate to the Online Snofyre Bug Report Form located at http://code.google.com/p/snofyre/issues/list.
    • If Snofyre is running, Click on Help > Report Bug . Snofyre will launch your browser and open the Online Snofyre Bug Report Form located at http://code.google.com/p/snofyre/issues/list.
    • Sometimes when Snofyre throws an internal error, it automatically opens up an Error Dialog with some error details filled in. In this case, clicking the Send Report button in the dialog will launch the browser with the prespecified URL.
      Tip: In this case Snofyre automically copies some of the error message stack to the System Clipboard, so you should be able to paste this as additional information when creating an error report.
  2. Please fill in the details of the bug/issue in the form that opens and click the Submit button on the form.
