Snofyre Installer First Steps

This section describes the initial steps of the Sappaheiros installation process.

Important: Please note that the installer for Snofyre is not the usual 'click Next' installer. It requirees the user to configure and install MySQL server on the client machine, as part of the installation. It is necessary to complete these steps for Snofyre to be properly configured for use. If these steps are not completed, then Snofyre might not start up or work as expected.
