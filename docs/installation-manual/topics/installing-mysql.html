Installing MySQL Server

Install MySQL server on your local machine using the downloaded file as described in the previous section. Please note that this section does not cover all the details of installing the MySQL Server. It assumes that the user knows how to download and install software. Please refer to the installation instructions on the MySQL site for your operating system, provided at http://dev.mysql.com/doc/refman/5.5/en/installing.html
Note: Please note that you'll require administrator rights on the client machine to install MySQL server. Please contact your local administrator if you do not have adequate rights on your machine.
Note: We recommend that you specify a password for the 'root' user account that MySQL sets up by default. We also recommend leaving the port used by MySQL Server to the default value (3306).
