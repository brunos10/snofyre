Running Snofyre

This section describes the running Snofyre on a client machine.

This chapter describes the process for running Snofyre on a client machine, after successful installation. If you require assistance with any of the steps described in this chapter, please contact Snofyre@googlegroups.com

Important: Please note that the user is expected to have configured and installed Snofyre as described in the Snofyre configuration manual. The configuration manual is located in the docs folder of the Installation Folder.
