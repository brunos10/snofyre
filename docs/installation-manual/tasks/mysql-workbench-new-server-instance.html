Creating a New Server Instance

This section describes the process for creating a new server instance for Snofyre

  1. Click on the New Server Instance button in the Server Administration section of the workbench home screen as shown below.

  2. In the dialog that opens up, select the option Take Parameters from Existing Database Connection, as shown below and click Next.

  3. MySQL workbench now displays a 'Testing the database connection' dialog as shown below. You must see the message 'Database connection test successfully', if you have followed the previous steps. Click Next>.

  4. MySQL Workbench will display the Testing Host Machine Settings dialog. It inform you that the 'MySQL configuration file does not exist', as shown below. You can ignore this error and click Next.

  5. In the next screen, you can review your MySQL Management parameters as shown below. You can change these parameters if you like, but for now the default values will do do just fine. Click on Next.

  6. In the next screen, enter the 'Server Instance Name' as snofyre@localhost, as shown below. Click Finish.

  7. You should now have a New Database Server Instance called snofyre@localhost displayed in the workbench home screen as shown below.
