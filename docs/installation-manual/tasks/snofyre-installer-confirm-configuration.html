Confirming Configuration in Installer

After the installer has copied the necessary files to the INSTALLATION_FOLDER, it presents a panel requesting the user to confirm MySQL server has been installer and configured properly (including data import).

In the panel presented, the user needs to confirm that all the required steps have been followed, by checking off against each task listed. The panel presented is shown below.
  • In the panel, confirm that you've followed all the steps by checking off each step as shown below and click Next.

  • If you click Next without checking all the boxes, the installer shows an error message as shown below.

If you have checked all the boxes and clicked on Next, the installer should take you to the next screen.
