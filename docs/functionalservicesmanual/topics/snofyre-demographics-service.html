Snofyre Demographics Model Module

This module provides contains a representation of all the Java classes that constitute the Demographics Model of Snofyre.

What is it?

This module provides contains a representation of all the Java classes that constitute the Demographics Model of Snofyre.

Why do you need it?

Most clinical queries invariably incorporate some aspect of the underlying demographics model (e.g. patient age, patient gender, etc). In order to support a sensible way to identify patients, Snofyre implements a simplistic Demongraphics Model to store data generated by the Snofyre Data Generator Service.
