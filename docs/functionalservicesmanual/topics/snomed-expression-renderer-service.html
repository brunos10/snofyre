SNOMED CT Expression Renderer Service

This module provides services for converting SnomedConcepts and Expressions, into their human readable forms.

What is it?

This module provides services for converting SnomedConcepts and Expressions, into their human readable forms.

Why do you need it?

The SNOMED CT Expression Converter Service provides means to generate the XML and Compositional Grammar for a given Expression. However, while both the forms are machine readable, they are not the most human readable representation of a given Expression. For this reason, SNOMED CT Expression Renderer Service provides a means for generating humam friendly labels for concepts and expressions.

This class implements the other methods for specifying rendering parameters. Please refer to the Javadocs provided for a complete list of methods.

Gotchas

Tip: HumanReadableRender has an implementation HumanReadableRendererImpl which is used to generate the rendering. This class can be tweaked to improve rendering if needed.
Important: HumanReadableRendererImpl uses the MRCMService to determine the human readable rendering. So the MRCMService must be available for this implementation to work.
