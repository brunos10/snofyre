Installation and Configuration of MySQL Workbench

This section describes the steps for downloading and installing an instance of MySQL Workbench on the client machine. Please note that this section does not cover all the details of installing the MySQL Workbench. It assumes that the user knows how to download and install software.
Note: Please note that you'll require administrator rights on the client machine to install MySQL Workbench. Please contact your local administrator if you do not have adequate rights on your machine.
