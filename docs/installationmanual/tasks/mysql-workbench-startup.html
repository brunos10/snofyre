Starting up MySQL Workbench

This section describes the process for starting up the MySQL Workbench, after installation.

This step varies based on the operating system.
  • For Windows XP, click the All Programs from Start. In the list that appear, click MySQL and then click on MySQL Workbench.
  • If you are a Mac OS X user, then double click MySQL Workbench from under Applications
The workbench should now start and appear on your screen as shown below.
