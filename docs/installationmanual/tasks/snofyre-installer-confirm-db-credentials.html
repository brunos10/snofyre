Confirming Database Credentials in Installer

After confirming that you have completed all the configuration steps, Snofyre installer prompts you to verify and specify the password for the snofyre user account on MySQL server. These are the username and password that you specified in previously in Creating a New User Account for Snofyre.

  1. The installer pre-populates some of the fields with default values, which are usually correct. In the panel presented (as shown below), the user needs to:
    1. Enter the snofyre user account on MySQL server.
    2. Enter the password for thesnofyre user account on MySQL server.
    3. Confirm that the default values presented are correct. They usually are, if you followed all the steps outlined in this manual.

  2. As shown below, in the Enter password field, specify the password you specified during the Creating a New User Account for Snofyre process. Click Next.

    If you have entered the correct password and clicked on Next, the installer should take you to the next screen.
    Important: If on the other hand, the password or any of the other details you have specified are not correct, the installer will display an error message as shown below. This usually happens, if you have not completed the configuration of MySQL properly. Please go back and check that you have followed all the steps in the previous chapters, if you have have an error.
